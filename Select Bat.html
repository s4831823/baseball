如何選球棒

喜歡從事棒壘球運動的朋友或許都有過買球棒的經驗，球棒分不同長度、重量、木頭材質、品牌，根據國外網站統計，多數MLB 球員使用的球棒品牌便多達 11 種以上，一支動輒好幾千元的球棒，要如何在琳瑯滿目的選擇中選擇一支最適合自己的呢？首先要先找到自己屬於何種型態的打擊者,以下有分為三種打擊型態的打擊者可以作為參考,可以參考這些型態的打擊者,找出自己適合使用什麼型號的球棒.

1.長打型球員

這類型球員簡單來說就是,擊到球時都會飛得很遠,長打率高但是打擊率會偏低,這類型球員的球棒甜蜜點地方都會比較大,擊球面積比較廣而棒頭會偏重如以下球員:

Khris Davis

2016年2月12日，Brewers將戴維斯交易到奧克蘭運動隊，為Jacob Nottingham和Bubba Derby。5月17日，在與德州遊騎兵隊的一場比賽中，戴維斯打出了3個本壘打，其中包括第三個擊球大滿貫。這是MLB歷史上第二次發生這種情況。

Joey Gallo

Gallo在Frisco開始2015賽季。6月1日，流浪者隊將Gallo晉升為主要聯賽。在他的第一場大聯盟比賽中，6月2日，蓋洛打出了他的第一個大聯盟本壘打並且開了4個跑（成為第一個在MLB首場比賽中實現四個打點的遊俠），在遊戲中打了3-4。

他們使用的球棒型號為C243,目前我們商品有AP5,Rizzo44這2款屬於這類型的球棒

Marucci Rizzo44

Rizzo44用於在單個Home Run Derby 中設置大多數本壘打的記錄，記錄設置Rizzo44的這種 變化更平衡且更容易擺動。

Marucci AP5

AP5用於在單個Home Run Derby 中設置大多數本壘打的記錄，記錄設置AP5的這種 變化更平衡且更容易擺動。

2.技巧型球員

這類型球員簡單來說就是,擊球點非常精準,長打率偏低打擊率較高,這類型球員的球棒甜蜜點地方都會比較小,控制球棒的技巧很好棒頭較輕較好控制如以下球員:

Dee Gordon

是美國職業棒球 外野手，游擊手和二壘手的西雅圖水手隊的棒球大聯盟（MLB）。他此前效力於洛杉磯道奇隊和邁阿密馬林隊隊。對於道奇隊，戈登主要是一個游擊手，而在馬林魚隊，他主要是二壘手。他開始打中鋒場在2018年與水手隊。2015年，在他與馬林魚隊的第一個賽季中，戈登隊的命中率為.333，共計205次命中，並偷走了58個基地。他在所有三個類別中領導了NL，並成為自1949年二壘手傑基羅賓遜以來第一個在平均擊球和被盜基地中領跑聯盟的NL球員。

鈴木一郎

Ichiro是第一位進入日本棒球名人堂（金球員俱樂部）的 MLB球員。他曾十次獲得美國職業棒球大聯盟全明星冠軍，並因三項賽事獲得2007年 全明星賽MVP獎，其中包括該項賽事首次在公園內全壘打。Ichiro 在他的大滿貫賽事的前十年贏得了Rawlings金手套獎，並且獲得了美國聯盟創造的七場連勝紀錄，其中有20場或更多場比賽，高達27場。他也因其長壽而聞名，繼續在2016年接近43歲時，擊球，擊打和基準百分比高於.300，產生高水平。在，Ichiro獲得了第3000名他的職業生涯MLB關閉的克里斯·魯辛的的科羅拉多洛磯山在庫爾斯球場，成為只有30球員曾經這樣做。他在日本和美國的職業生涯總共有超過4300次點擊。

他們使用的球棒型號為P72,目前我們商品有PS2這款屬於這類型的球棒

Sam bat PS2

Sam bat PS2用於在單個Home Run Derby 中設置大多數本壘打的記錄，記錄設置Sam bat PS2的這種 變化更平衡且更容易擺動。

3.平均型球員

這類型球員簡單來說就是,擊到球時都會飛得很遠,長打率高擊球準確度高,球棒棒頭比長打型球員球棒稍小如以下球員:

Mike Trout

是美國職業棒球 外野手為洛杉磯天使的職業棒球大聯盟（MLB）。特勞特是七屆美國職業棒球大聯盟全明星，並在2014年和2016年獲得美國聯盟（AL）最有價值球員（MVP）獎（在2012年，2013年和2015年的投票中獲得第二名）。

J.D. Martinez

是美國 職業棒球 外野手和指定打擊的波士頓紅襪隊的棒球大聯盟（MLB）。他還曾在休斯敦太空人隊，底特律老虎隊和亞利桑那響尾蛇隊效力於美國職業棒球大聯盟。馬丁內斯是右手投擲者和擊球手

他們使用的球棒型號為C271,目前我們商品有MT27,JD28這2款屬於這類型的球棒
